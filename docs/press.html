Short (Twitter / socials)

A 1.6 MB native menu bar clipboard manager for Mac, Windows and Linux. Clipboard history in a native dropdown, keep-awake sessions, optional date/temperature next to the icon. Pure Rust, no Electron, no telemetry. €1.99 one-time or 7-day free trial.

Long (blogs / newsletters)

klipa is a tiny, fast menu bar clipboard manager for macOS, Windows and Linux. It keeps a history of everything you copy - text and images - in a native menu bar dropdown; click any entry to copy it back and paste. It also offers keep-awake sessions to stop your machine from sleeping (5 minutes to 5 hours, or indefinitely), and can optionally show the date, the current temperature at your location, or both next to the clipboard icon in the status bar. klipa is about 1.6 MB, written in pure Rust with no Electron, no WebView and no JavaScript runtime. Your clipboard history stays on your device - klipa has no accounts, no cloud sync and no telemetry. It's MIT-licensed and available for €1.99 one-time (with a 7-day free trial) direct-download - licenses sold on Gumroad and Ko-fi - plus on the Mac App Store, Homebrew, Scoop, winget and AUR.

Feature bullets (for lists / reviews)

  • Native menu bar dropdown - clipboard history one click away, no window, no dock icon.
  • Text and image entries - screenshots get a preview thumbnail in the dropdown.
  • Keep-awake sessions - 5 min to 5 hours, or indefinitely (uses caffeinate / SetThreadExecutionState / systemd-inhibit).
  • Menu bar display - swap the icon for today's date, the temperature at your location, or both.
  • Configurable dropdown size - show 10, 25, 50 or 100 recent entries.
  • Ultralight - ~1.6 MB binary, ~10 MB RAM. No Electron, no browser engine.
  • Local-only by default - no accounts, no telemetry, no cloud.
  • Cross-platform - macOS 11+, Windows 10/11, Linux (x86-64). One Rust codebase.
  • Open source - MIT-licensed at github.com/peterdsp/klipa.

Install commands (copy-paste)

macOS - Homebrew

brew tap peterdsp/klipa https://github.com/peterdsp/klipa
brew install --cask klipa

Windows - winget or Scoop

winget install dev.peterdsp.klipa
scoop bucket add klipa https://github.com/peterdsp/klipa
scoop install klipa

Linux - Arch (AUR)

yay -S klipa-bin

Any platform - from source

cargo install --git https://github.com/peterdsp/klipa klipa-ui
