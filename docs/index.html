Why klipa

Tiny footprint

A single self-contained binary that sits around 30–50 MB RSS — no Electron, no browser engine.

Unlimited history

Every copy is persisted to SQLite and kept. Search it instantly, fuzzy or exact.

Keyboard-first

Summon with ⌘⇧V / Ctrl+Shift+V. Arrow keys, Enter to paste, Esc to dismiss.

Lives in the tray

No dock icon, no taskbar clutter. klipa stays out of the way until you need it.

Truly cross-platform

The same Rust core on macOS, Windows, and Linux — GPU-accelerated Slint UI.

Open source

MIT licensed. Read it, build it, ship it. View on GitHub.
